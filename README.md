# SSH_Util

## Features:
```
Utility to connect to hosts via SSH via URL link for Windows
```

## To get started::
* **Download the latest revision**
```
git clone https://github.com/VHSgunzo/SSH_Util.git
```
* **From the utility folder**
```
1. Copy putty_util.exe or putty_util.bat to C:\Program Files\PuTTY\
2. Apply the appropriate file for the registry ssh(for_EXE).reg or ssh(for_BAT).reg
```

## Notes:
* The system must have PuTTY installed
* The URL link format should be like ssh://8.8.8.8 or ssh://login@8.8.8.8
